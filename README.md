# Difference Calculator

Runs from the command line, compares two configuration files and shows a difference.
Working with JSON and YAML.

## Usage

**help:**

```bash
$ gendiff -h | --help
```

**Running the script with default settings:**

```bash
$ gendiff <file_path1> <file_path2>
```
